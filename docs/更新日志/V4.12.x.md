#### 2. 执行升级脚本

该脚本仅需商业版用户执行。

从任意终端，发起 1 个 HTTP 请求。其中 `{{rootkey}}` 替换成环境变量里的 `rootkey`；`{{host}}` 替换成FastGPT域名。

```bash
curl -X POST "{{host}}/api/admin/initv412" \
  -H "Content-Type: application/json" \
  -d '{"rootkey": "{{rootkey}}"}'
```
